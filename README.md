# nextjs-generic-landing-page

A generic landing page template, built in Next.js with static HTML export.

- Title/Tagline
- Signup form
- Big image
- Big video
- Try the app (interactive)
- 3 features
- 3 testimonials
- Pricing
- Login link
- Footer


## How to use

Run:

    yarn dev

Export static HTML:

    yarn export

Test the static HTML:

    cd out
    python -m SimpleHTTPServer
