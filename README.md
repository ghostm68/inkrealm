# nextjs-generic-landing-page

A generic landing page template.

- Big image
- Big video
- Signup form
- Try the app (interactive)
- Tagline
- 3 features
- 3 testimonials
- Pricing
- Footer
- Login link


## How to use

Run:

  yarn dev

Export static HTML:

  yarn export

Test the static HTML:

  cd out
  python -m SimpleHTTPServer
